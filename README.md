## 项目功能

本项目是一个罗汉果花识别分类系统，主要功能包括：

1. 图片上传：支持用户上传罗汉果花的图片
2. 图片预览：实时预览上传的图片
3. 自动识别：识别罗汉果花的生长阶段
4. 结果展示：展示识别结果

## 系统功能

本系统是基于深度学习的罗汉果花智能识别分类系统，主要功能包括：

### 核心功能
- 智能识别：采用深度学习模型对罗汉果花进行识别和分类
- 结果分析：提供详细的识别结果，包括生长阶段、置信度等
- 实时预览：支持图片实时预览和快速识别

### 技术特点
- 现代化界面：采用Material-UI设计规范
- 响应式布局：支持各种设备屏幕
- 用户友好：操作简单直观，反馈及时

## 使用说明

1. 通过点击上传区域或拖拽方式上传罗汉果花图片
2. 系统自动预览上传的图片
3. 点击"开始识别"按钮进行智能识别
4. 查看详细的识别结果和分析报告
